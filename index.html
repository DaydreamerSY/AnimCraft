Help & Cheat Sheet

How to Use AnimCraft

Add Elements

Click Button or Item from the top-left toolbar to place a shape on the canvas. Drag shapes to reposition them.

Inspector Panel

Click a shape to open the Inspector. Edit position, scale, opacity, color, and name. Use Duplicate to clone.

Animation Sequences

Click + Add Animation in the Inspector. Choose type (Move, Scale, Fade, Bounce), set duration and Easing Curve using the Transition + Direction dropdowns (Godot style).

Curved Paths

Move animations show a trajectory line on canvas. Drag the yellow control point to create Bezier curves. Dashed lines show overshoot/anticipation for Back/Elastic eases.

Playback

Click Play in the topbar to run all animations by Sequence Step order. Click ▶ on individual steps to preview one animation.

Import / Export

Use Export JSON to save your scene. Use Import JSON to load from file or paste JSON data to restore a scene.

Easing Curve Reference

Transition: Linear, Sine, Quad, Cubic, Quart, Quint, Expo, Circ, Back, Elastic, Bounce, Spring
Direction: In (accelerate), Out (decelerate), InOut (both), OutIn (reverse both)
